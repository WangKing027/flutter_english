# flutter_english

一个模仿流利说线上学习英语的app

## Getting Started

项目亮点：
1. 调用AudioPlayer播放assets和本地文件夹/网络资源音频,封装成AudioPlayerFactory调用
2. 封装EventChannel/MethodChannel，通过with 实现重写方法实现具体页面逻辑具体处理
3. 模仿流利说的打分显示不同音效
4. 动画组件封装，hero动画，路由动画封装
5. 项目结构封装，封装Provider进行状态管理
